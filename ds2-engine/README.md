# DS2 Engine (app folder)

This folder is the React app. **Start here for full instructions:** see the [README in the parent folder](../README.md).

Quick start:

```bash
npm install
npm run dev
```

Open http://localhost:5173

Scripts:

| Command | What it does |
|---------|----------------|
| `npm run dev` | Run the app while you develop |
| `npm run build` | Build static files into `dist/` |
| `npm run preview` | Preview the built version locally |
